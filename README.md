## 📈 학습 목표

1. **클라우드 환경 이해**:
   - AWS 핵심 서비스의 생성, 설정, 관리.
2. **운영체제 관리 역량 강화**:
   - 리눅스 서버의 계정, 프로세스, 서비스 관리.
3. **IaC 활용 능력**:
   - Terraform을 통한 인프라 코드화.
4. **CI/CD 구현 능력**:
   - 배포 자동화 파이프라인 설계 및 운영.
5. **보안 및 비용 최적화 고려**:
   - IAM 권한 설계, 리소스 관리 비용 분석.
6. **협업 및 코드 관리 역량**:
   - Git/GitHub을 통한 협업 워크플로우 경험.
